    • Favicones


    Un favicon es la pequeña imagen que se muestra en la pestaña del navegador o en la lista de marcadores (favoritos). En la barra de dirección, el tamaño del favicon es bastante reducido, 16x16 píxeles, pero en otros lugares como los marcadores puede tener un tamaño mayor (24x24, 32x32, 48x48 o 64x64. .


    Procedimiento para ser aplicado a la página web


    Este código tiene que ir en la parte del head de nuestro sitio web, es decir, después de la apertura de la etiqueta < head> y antes de su cierre < /head>.

    La imagen que debes utilizar debe estar en formato icon para ello debes hacer la conversión de un formato png a formato ico, para ello mira los pasos:



    1. Descarga la imagen en formato png que desees utilizar en tu página web.


    2. Ingresa a la página https://convertico.com



    3. El realiza la conversión respectiva y luego clic en descargar



    4. Luego lleva la imagen que convirtió a la carpeta IMG de tu proyecto


    5. Aplica el código en el < HEAD> utilizando la etiqueta de imagen, mira el ejemplo:

    6. Guarda el archivo, abre la página web nuevamente y observa como aplicó el favicón
