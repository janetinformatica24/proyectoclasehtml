    Definición: Una etiqueta es una marca con clase que delimita una región en los lenguajes basados en XML.


    La Web se basa en el HTML, o «lenguaje de marcado de hipertexto», que está basado en el uso de etiquetas. Las etiquetas (entre otras muchas cosas) le dicen al programa visualizador de páginas web (o navegador) en qué juego de caracteres está la página, de qué tipo es cada uno de los fragmentos de texto que contiene (por ejemplo, encabezamiento, texto normal, etc.), si están alineados a un lado o centrados, en qué tipo de letra está el texto (cursiva, negrita, etc.), si hay tablas, de qué anchura son etc. Dicho de otro modo: las etiquetas dan al navegador las instrucciones necesarias para que presente la página en pantalla.


    Las etiquetas son palabras claves que encerramos en los signos "MENOR QUE" y "MAYOR QUE": < >, cada una de ellas contiene un Atributo: (Es el nombre que se les da a las características, propiedades que cualifica las etiquetas).


    Algunas etiquetas


    !DOCTYPE html



    < !DOCTYPE html>DOCTYPE es la declaración de tipo de documento. En otras palabras, el doctype nos sirve para indicar que nuestro documento está escrito siguiendo la estructura determinada



    Head



    < head>ESCRIBO INFORMACION TÉCNICA< /head>

    < Head > ** < /head>: Información Técnica para el navegador: El head es la parte donde se incluye la información acerca del documento.



    Meta charset



    Meta charset=“UTF-8”: Añade información a la página que es útil para los buscadores. También permite definir los caracteres (charset) que se trabaja en el sitio. Es un estandar de la codificación UTF-8 y es compatible con todos los navegadores. No necesita cierre.


    Es importante que el meta charset se coloque justo después de la apertura de la etiqueta < head>. De este modo nos aseguramos que el navegador conocerá la codificación de caracteres utilizada antes de que comience a procesar nada.



    Title



    Título de la Página. Es lo que voy visualizar en la barra de título o en la pestaña del navegador. Es una etiqueta par y se aplica asì:


    < title> Titulo de la Página < /title>



    Html



    Html lang=“es” : Inicio del documento con atributo: lang de language, "es"= de español. (Etiqueta par), su aplicación sería así:

    < html lang="es"> < /html> Es la última etiqueta que se cierra.



    Body



    Body: Aquí es donde va todo el contenido de la página, lo que el usuario visualizará y dentro de este se aplican las demas etiquetas para darle presentación a nuestro página. Ejemplo


    < body background="nombrearchivo.gif">: para fondo de imagen < /body>


    < body bgcolor="color">: para color de fondo < /body>



    Etiqueta h1



    < h1 >** < /h1>: Es el elemento HTML utilizado, de manera habitual, para identificar la cabecera más importante en una página web. Ejemplos de H1 serían el título de una página, el título de un post, el nombre de un producto. El lenguaje HTML nos permite establecer diferentes cabeceras según su importancia, es asì que dependiendo de la necesidad se aplican las etiquetas H1, H2, siendo h2 para subtìtulos.

    Listas de definiciones


    Listas de definiciones con etiquetas < dl>, < dt> y < dd>


    Si lo que vamos a hacer es un listado de definiciones, podemos usar las etiquetas < dl>, < dt> y < dd>. Vamos a explicarlas por partes:



    Etiqueta DT


    La etiqueta < dt> viene de los términos “Definition term” y dentro de ella irá el término que vamos a definir. Para entendernos mejor, dentro de < dt> iría el título de la definición.< /dt>



    Etiqueta DD


    La etiqueta < dd> viene de los términos “Definition description” y nos dice que dentro de ésta irá la definición.


    Si escribimos varios listados de definición, éstas se separarán automáticamente entre ellas para facilitar su diferenciación.Observa el ejemplo


    Aquí va el término que definiremos
    Y aquí dentro irá la definición propiamente dicha.

    Aquí va la segunda definición
    Y aquí dentro irá la segunda definición, separada automáticamente de la anterior.
